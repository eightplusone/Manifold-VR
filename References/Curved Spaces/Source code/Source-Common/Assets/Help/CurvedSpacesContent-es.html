Cómo elegir el contenido

Usa el menú Vista para seleccionar qué objetos quieres en la escena.

  • Pieza central
  • Colorear las caras asociadas
  • Paralelas de Clifford
  • Figuras de vértice

Las flechas del teclado (y no el menú Vista) controlan los muros. Una otra página describe cómo retirar los muros.
