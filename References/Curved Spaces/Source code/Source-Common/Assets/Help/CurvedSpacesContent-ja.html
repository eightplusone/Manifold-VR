作画内容設定

「表示」メニューより、Curved Spaces が、何を作画するのか設定します：

  • 中央物体
  • 対応壁対の色分け
  • クリフォード平行線
  • 頂点図形

壁開口の大きさは「表示」メニューからではなく、カーソル左右移動キーを使って、調節します。詳細は「壁開口のサイズ調節」ページにあります。
