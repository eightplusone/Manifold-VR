中心物体

尽管地球和星系看起来是最漂亮的，但是地球和陀螺仪却能最好地显示其周围胞腔的相对定向。举个例子来说，通过地球和陀螺仪你能够很容易地告诉人们其周围的胞腔的方向是相反的，或者是相差180°｡　本软件中的星系图像是依据由哈勃太空望远镜抓取的涡旋星系 Messier 51 (M51, NGC 5194)的图像制作的。
