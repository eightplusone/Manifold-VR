中心物體

儘管地球和星系看起來是最漂亮的，但是地球和陀螺儀卻能最好地顯示其周圍胞腔的相對定向。舉個例子來說，通過地球和陀螺儀你能夠很容易地告訴人們其周圍的胞腔的方向是相反的，或者是相差180°｡　本軟件中的星系圖像是依據由哈勃太空望遠鏡抓取的渦旋星系 Messier 51 (M51, NGC 5194)的圖像製作的。
