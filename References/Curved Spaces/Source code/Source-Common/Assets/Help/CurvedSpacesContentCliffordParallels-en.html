Clifford Parallels

Clifford parallels fill up the 3D surface of a hypersphere the same way that strands of spaghetti fill up the box they’re in. Each Clifford parallel is a great circle, yet any two of them twist around each other like two linked rings.

The Clifford Parallels option is available only for the hypersphere (3-sphere).
