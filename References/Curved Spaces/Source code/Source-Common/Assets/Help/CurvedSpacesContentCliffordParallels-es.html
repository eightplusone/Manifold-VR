Preferencias de contenido

Paralelas de Clifford

Las paralelas de Clifford llenan la superficie tridimensional de una hiperesfera de la misma manera que los espaguetis llenan la caja en la que los compras. Cada paralela de Clifford es un gran círculo, sin embargo cada par de ellos se tuerce que las dos paralelas se cruzan como dos anillos entrelazados.

La opción de las paralelas de Clifford se puede activar sólo para la hiperesfera (3‑esfera).
