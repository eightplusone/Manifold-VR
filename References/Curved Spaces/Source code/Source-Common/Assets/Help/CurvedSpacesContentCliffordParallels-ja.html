クリフォード平行線

クリフォード平行線は、３次元超球（3-sphere）でのみ、表示できます。

クリフォード平行線は、ちょうど真っ直ぐなスパゲティが、直方体の箱を隙間無く満たすように、超球の３次元表面を、隙間無く満たします。クリフォード平行線は、それぞれが大円ですが、どの２本を取ってきても、互いに相手の周囲に巻き付いていて、ちょうど鎖の輪同士が、つながっているようになっています。
