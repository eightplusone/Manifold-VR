内容选择

克利福德(Clifford)平行线

类似于在纸盒中盛放意大利面，克利福德平行线填充了整个超球面。其中每一条克利福德平行线都是一个大圆。它们彼此间像二链环一样缠绕着。

菜单栏中的克利福德平行线选项只在超球面上可用（三维球面）。
