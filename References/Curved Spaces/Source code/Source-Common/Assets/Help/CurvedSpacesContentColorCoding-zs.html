彩色编码

对一个多面体沿着它的面成对相粘可以构造一个有限的宇宙空间。利用彩色编码选项赋予成对的相粘面相同的颜色有助于你看清楚多面体如何沿着面相粘。

在比较不常见的带镜像面的多面体上，每个多面体的面都是自相粘的，因此都被单独赋予了一种颜色。
