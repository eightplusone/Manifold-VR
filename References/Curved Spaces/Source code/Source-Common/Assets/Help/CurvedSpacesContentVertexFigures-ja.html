頂点図形

「端のない有限宇宙」を、多面体から生成したとき、多面体の頂点は、それぞれの宇宙空間特有の対称性を持って、合わさっています。例として 3-Torus を見てみましょう。「空間」メニューから「別の空間を選択する…」を選択し、Basic フォルダー内の 3-Torus.gen を開いてください。3-Torus は、立方体（正６面体）からできていますが、「頂点図形」をオンにすると、頂点が、正８面体の対称性をもって、合わさっていることがわかります。

頂点図形がはっきりと見えるようにするには、カーソル左右移動キー（ ← と → ）を使って、壁開口を、多面体の稜（辺）のみが細く残るように、大きく開いてください。壁に邪魔されずに、頂点図形の全体像を見ることができます。
