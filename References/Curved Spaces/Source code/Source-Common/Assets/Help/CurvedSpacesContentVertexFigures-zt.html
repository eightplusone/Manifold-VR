頂點圖像

當利用一個多面體構造一個有界空間時，你會發現多面體的拐角部分以它們特有的對稱方式粘合在一起。 以三環面為例，首先點擊菜單欄里的空間，點擊選取別的空間，最後在Basic文件夾中選取3-Torus。 那麼你會發現在立方體粘成3-torus的同時，拐角部分粘合成了一個八面體。

想看清頂點圖像，請使用鍵盤的方向左鍵（ ← ）和右鍵（ → ）來調整牆體的大小至非常窄。這樣你的視線就可以覆蓋整個圖像而不被牆體阻擋。
