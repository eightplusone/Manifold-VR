How to export the image

Save

Choose Export ▸ Save Image… to save the current image as a TIFF file (Macintosh) or a BMP file (Windows).

Copy

Choose Export ▸ Copy Image to copy the current image in TIFF format (Macintosh) or in BMP format (Windows). You can then paste the image into your favorite word processor or graphics program.

Print

Curved Spaces does not support printing directly. If you’d like to print an image, please copy-and-paste the image into your favorite word processor or graphics program as explained above and print it from there.
