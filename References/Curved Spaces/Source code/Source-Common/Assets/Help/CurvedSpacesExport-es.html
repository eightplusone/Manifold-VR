Cómo exportar la imagen

Guardar

Elige Exportar ▸ Guardar imagen… para guardar la imagen actual como archivo TIFF (Macintosh) o como archivo BMP (Windows).

Copiar

Elige Exportar ▸ Copiar imagen para guardar la imagen actual como archivo TIFF (Macintosh) o como archivo BMP (Windows). Posteriormente puedes pegar la imagen en tu procesador de textos o imágenes favorito.

Imprimir

Curved Spaces no tiene soporte para imprimir las imágenes directamente. Si quieres imprimir una imagen, por favor copia y pégala en tu procesador de textos o imágenes, tal y como se explica más arriba, e imprímelo desde allí.
