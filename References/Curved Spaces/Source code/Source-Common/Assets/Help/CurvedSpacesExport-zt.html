如何導出圖像

存儲

選擇 導出 ▸ 存儲圖像… 在Mac系統下保存為TIFF文件，在Windows系統下保存為BMP文件。

拷貝

選擇 導出 ▸ 拷貝圖像 在Mac系統下以TIFF格式拷貝當前圖像，在Windows系統下以BMP格式拷貝文件。然後您可以選擇在文字軟件或者圖形圖像軟件中粘貼。

打印

本軟件目前不支持直接打印。如果您想打印圖像，請按照上面步驟複製粘貼在文字軟件或者圖形圖像軟件中再打印。
