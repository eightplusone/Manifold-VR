宇宙空間設定

「空間」▸「別の空間を選択する…」を選択し、Sample Spaces フォルダー内の任意のファイルを開いてください。Basic サブフォルダー内には、とりわけ単純で、美しい空間の例が入っているので、それらの空間から、探険すると良いでしょう。Spherical、Flat、そして Hyperbolic のサブフォルダーには、よりヴァラエティーに富んだ諸空間が、その曲率によって分類されています。

上級者は、自分で新しい空間を定義することができます。
