如何選取空間

點擊 空間 ▸ 別的空間選取… 然後選取Sample Spaces文件夾中的任何文件。其中Basic子文件夾中已經包含一些特殊且漂亮的空間，Spherical, Flat 和 Hyperbolic 子文件夾中包含了更多的特殊曲率的空間。

如想創作自己的空間，可參考創作自己的空間。
