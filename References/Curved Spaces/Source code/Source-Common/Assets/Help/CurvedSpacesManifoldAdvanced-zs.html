如何创建空间
（仅针对三维流形专家）

该软件不仅限于运行已有的样本空间。只需要创建一个简单的包含生成空间的复叠变换群的一列４×４矩阵的.txt文件。参考

  • 三环面,
  • 庞加莱(Poincaré)十二面体空间，和
  • 带镜像的直角十二面体.
球面空间对应的矩阵是群O(4)元素。双曲空间对应的矩阵是群O(3,1)的元素。平坦空间对应的矩阵是相对应的群的元素。

本软件采用的是　(行向量)(矩阵)　而不是　(矩阵)(列向量)。因此运行的矩阵可能是你所期望的矩阵的转秩。当然在球面空间里，由于运行的矩阵等于你所期望的矩阵的逆，因此它们没有区别。但是在平坦和双曲空间里，矩阵的准确书写是非常重要的。

首先在记事本里保存4×4矩阵为一个简单的.txt文件。然后将文件的格式改为.gen。如果要运行刚才保存的文件，请先把.gen文件放入Sample Spaces文件夹，然后在软件中点击空间 ▸ 别的空间选取…。

Mac系统: 想读取Sample Spaces的文件，请先右击本软件的图标选择显示包内容，然后点击Contents/Resources/Sample Spaces。

相比于弯曲空间1.0版，弯曲空间3.0版本包含了更少，更精简的样本空间。如想得到其他的空间请点击与我联系。
