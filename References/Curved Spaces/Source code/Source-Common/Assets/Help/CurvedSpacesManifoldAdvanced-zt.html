如何創建空間
（僅針對三維流形專家）

該軟件不僅限於運行已有的樣本空間。只需要創建一個簡單的包含生成空間的復疊變換群的一列４×４矩陣的.txt文件。參考

  • 三環面,
  • 龐加萊(Poincaré)十二面體空間，和
  • 帶鏡像的直角十二面體.
球面空間對應的矩陣是群O(4)元素。雙曲空間對應的矩陣是群O(3,1)的元素。平坦空間對應的矩陣是相對應的群的元素。

本軟件採用的是　(行向量)(矩陣)　而不是　(矩陣)(列向量)。因此運行的矩陣可能是你所期望的矩陣的轉秩。當然在球面空間里，由於運行的矩陣等於你所期望的矩陣的逆，因此它們沒有區別。但是在平坦和雙曲空間里，矩陣的準確書寫是非常重要的。

首先在記事本里保存4×4矩陣為一個簡單的.txt文件。然後將文件的格式改為.gen。如果要運行剛才保存的文件，請先把.gen文件放入Sample Spaces文件夾，然後在軟件中點擊空間 ▸ 別的空間選取…。

Mac系統: 想讀取Sample Spaces的文件，請先右擊本軟件的圖標選擇顯示包內容，然後點擊Contents/Resources/Sample Spaces。

相比於彎曲空間1.0版，彎曲空間3.0版本包含了更少，更精簡的樣本空間。如想得到其他的空間請點擊與我聯繫。
