如何操控飛船

  1. 在場景里點擊鼠標左鍵，然後操控鼠標。再次點擊鼠標左鍵退回至光標。
  2. 按鍵盤（ ↑ ）方向鍵加速。 按鍵盤（ ↓ ）方向鍵減速。 按空格鍵暫停。

    如果有觸摸板，你可以有兩根手指頭在觸摸板上滑動以控制速度（需觸摸板支持滾動）。

    如果有帶滾輪的鼠標，你可以用滾輪來控制速度。

  3. 按空格鍵暫停，在場景中點擊鼠標左鍵，然後拖動鼠標
    • 不按任何鍵
    • 按住Ctrl鍵
    • 按住Shift鍵
    • 同時按住Ctrl鍵和Shift鍵
    以體驗完全6維自由度的運動。
