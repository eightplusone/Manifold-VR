Comment rétracter les murs

  1. Appuyez sur la touche fléchée à gauche ( ← ) pour élargir l'ouverture.
  2. Appuyez sur la touche fléchée à droite ( → ) pour rétrécir l'ouverture.

Quand l'ouverture est totalement fermée vous voyez la cellule fondamentale à partir de laquelle cet espace est construit. Quand l'ouverture est presque totale, vous avez l'illusion de voir un espace plus grand, pavé par la répétition de la cellule fondamentale. Cette illusion provient du fait que, dans un univers fini, votre ligne de visée se reboucle sur elle-même vous permettant de vous voir par derrière, par les côtés, par le dessus, etc.
