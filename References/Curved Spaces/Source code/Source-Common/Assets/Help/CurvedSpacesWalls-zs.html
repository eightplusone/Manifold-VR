如何缩进墙面

  1. 按住方向键左键（ ← ）扩大孔洞。
  2. 按住方向键右键（ → ）缩小孔洞。

当孔洞完全关闭的时候，你可以很清楚的看到构成这个空间的基本胞腔。当孔洞接近最大的时候，你可以看到更大的由重复的基本胞腔粘接得到的空间。之所以能够看到这样的景象，那是因为视线会在有限的宇宙里环绕有助于你能够从后面，侧面，上面等等看到自己。
