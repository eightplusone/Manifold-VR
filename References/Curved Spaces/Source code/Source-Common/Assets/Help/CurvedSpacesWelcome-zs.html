弯曲空间

弯曲空间是一个关于有限宇宙空间模型的飞行模拟器，或者说用于模拟构造三维的球面，平坦和双曲空间。想知道如何操作该软件，请先阅读位于页面左侧的如何操控飞船。

上图为在有镜像墙和纯直角的双曲十二面体中的视图。其中单一的星系坐落在十二面体的中心。或者，你也可以把上图解释为由无限多个一样的直角十二面体平铺的一个三维双曲空间。不管怎么说，上图有点特殊。但是在文件夹中的大部分样本空间所表示的宇宙都是由不含有反射的环绕相粘。
